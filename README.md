# Elite GS Landing Page

A modern and responsive landing page for **Elite GS**, a men's clothing store.

## About the Project

This project was created to showcase a clean and professional website for a men's fashion brand. The goal is to provide visitors with information about the store, its products, and contact options.

## Features

* Responsive design
* Modern and minimal layout
* Hero section
* About section
* Product showcase
* Contact section
* Clean HTML and CSS structure

## Technologies Used

* HTML5
* CSS3
* Git
* GitHub
